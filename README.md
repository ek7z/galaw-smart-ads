# Galaw Smart Advertising Landing Page

A premium, high-converting, and modern landing page built for **Galaw Smart Advertising**, featuring smooth scrolling, interactive animations, a dynamic lead capture form, and custom Gmail SMTP mailer integration.

---

## 🚀 Features

- **Rich Modern UI/UX:** Responsive layout with smooth gradients, dark mode, hover effects, and staggered animations (via AOS).
- **Interactive Ambient Mouse Glow:** Premium desktop mouse-tracking glow.
- **Dynamic Typewriter Headline:** Automatically cycles through key business value-propositions.
- **Responsive Slider/Carousel:** High-fidelity customer testimonial carousel (fully responsive).
- **Gmail SMTP Lead Form:** Submits contact details directly to a configured Gmail inbox using SMTP (via PHPMailer or Express Node.js).
- **Client Hand-off Guide:** Includes a ready-to-use [SMTP_GUIDE.md](./SMTP_GUIDE.md) to help clients plug in their own credentials easily.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, Tailwind CSS (via CDN), Vanilla JavaScript
- **Backend Options:**
  - **PHP:** `send_email.php` (utilizing PHPMailer)
  - **Node.js:** `server.js` (utilizing Express & Nodemailer)
